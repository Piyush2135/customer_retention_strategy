# Decoding Customer Value: A SQL-Driven Retention Strategy

This project was developed as part of IIT Guwahati Summer Projects '26.

## Objective
Analyze customer transactional and behavioral data to identify high-value customers, evaluate promotional dependency, and design a data-driven retention strategy.

## Project Components
- Raw dataset
- Cleaned dataset
- Engineered features
- SQL database
- Jupyter notebook
- SQL query outputs
- Founder dashboard
- Executive summary
- Retention playbook

## Tools Used
Python | Pandas | SQLite | SQL | Power BI
